
# Orkestra project template

This template is meant to help you scaffold a new Orkestra project.

## Requirements

### [copier][copier]

```bash
brew install pipx
pipx install copier
```

## Usage

```bash
copier gh:knowsuchagency/orkestra_template destination_directory
```


[copier]: https://copier.readthedocs.io/en/latest/
